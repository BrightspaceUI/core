## Best Practices
<!-- docs: start best practices -->
<!-- docs: start dos -->
:white_check_mark: Use buttons to cause an action or launch a workflow  
:white_check_mark: Keep button text short - see "Writing" guidelines
<!-- docs: end dos -->

<!-- docs: start donts -->
:x: Don't use multiple Primary buttons - the primary action should always be clear and obvious  
:x: Avoid icon-only buttons with unfamiliar icons, only use these for familiar concepts  
:x: Don't use buttons for navigation, use a [Link](../link/README.md) instead  
:x: Don't open menus with buttons - use a [Dropdown](../dropdown/README.md) instead
<!-- docs: end donts -->
<!-- docs: end best practices -->
